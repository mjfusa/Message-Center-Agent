Secrets:

AZURE_CREDENTIALS: Contains the Azure Service Principal credentials in JSON format.
M365_TENANT_ID: The Microsoft 365 tenant ID.
CLIENT_ID: The client ID of the app registration in Microsoft Entra.
Pre-requisites:

Ensure the Teams Toolkit CLI is installed and configured in your project.
The Azure Service Principal must have sufficient permissions to create app registrations and grant admin consent.
Provision and Deployment:

The provision step handles the Entra app registration and Teams Developer Portal setup.
The deploy step publishes the app to Microsoft Teams.
Validation:

The validate step ensures the app is correctly deployed and functional.
